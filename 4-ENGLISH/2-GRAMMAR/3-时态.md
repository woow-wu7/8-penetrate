# 时态

- 时态 和 ( 动词 ) 有关系
- 动词随着时间改变形态

# (一) 现在简单式

```
1. 动词的形式
- be动词: 强调状态，存在
  - am
  - is
  - are
- 一般动词: 强调动作
  - 如果主词是第三人称单数时，动词+s

2. 时间副词 adverb
  - now
  - every + 时间: 每一个什么什么
```

### (1.1)使用的时机

```
1
表示现在的状态或动作
---
There are many visitors in the zoo. 动物园有很多游客
Here comes the bus. 公共汽车来了

```

```
2
表示现在习惯性的动作
---
David often sleeps during class.
My parents take exercise in the park every morning.
// exercise 运动 -- 不加s
// exercise 习题 -- math exercises
// take exercise = play sports // play a sport
```

```
3
表示不变的事实或真理
---
The earth moves around the sun.
```

# (二) 过去简单式

```
1. 动词的形式
- be动词: 强调状态，存在
  - was
  - were
- 一般动词: 强调动作
  - 过去式动词 ( 不分人称，+ed )
    - 规则变化: + ed
    - 不规则变化
      - run -> ran

2. 时间副词 adverb
  - yesterday
  - the day before yesterday 前天
  - last + 时间
  - last night
  - last week
  - ( 时间 + ago )
  - ( before )
  - ( then: 那是 )
```

### (2.1)使用的时机

```
1
表示过去的动作或状态
---
I bought this yesterday.
There was an old temple over there.
```

```
2
表示过去习惯性的动作
used to + 原型动词
[ used to 曾经 ]
My father used to smoke, but now he doesn't.
```

# (三) 现在进行式

```
1. 动词的形式
- be动词: 强调状态，存在
  - am + Ving
  - is
  - are

2.1
原形动词 ----------------------> ing
talk - talking
say - saying
speak - speaking

2.2
原形动词字尾有e ----------------> 去掉e + ing
have -> having
write -> writing
come -> coming

2.3
原形动词为 子音 + 短母语 + 子音 --> 重复字尾 + ing
a e i o u
put - putting
cut - cutting
swim - swimming
we are eating breakfast now. 现在进行时
we ate breakfast before going to school. 一般过去时
we eat breakfast every morning. 一般现在时
```

### (3.1)使用的时机

```
1
表示现在正在进行的动作
Jon is watching the baseball game on TV now.

2
表示重复发生的动作，常常伴随 always
- 1. always
- 2. all the time
- 3. agin and agin
-- He is always complaining ( about + 受词 ).
-- The car is breaking down all the time.
-- always === all the time 总是


3
表示最近的未来即将发生的动作，此法常用于某些动词
- 比如(来去动词)可以用现在进行时来表示未来: come go start leave arrive 等
-- I'm leaving for china tomorrow.
  - leave china 离开中国
  - leave for china 来到中国
-- My boyfriend is coming to see me this afternoon.
```

```
某些动词不可以用在进行时中
- 表示感官的动词 sea hear smell
- 表示情感的动词 love like
- 其他 have know
-- i'm seeing the bird in the tree ------- (错)
-- i'm looking at the bird in the tree --- (对)
-- hear - listening
-- sea - looking
```
