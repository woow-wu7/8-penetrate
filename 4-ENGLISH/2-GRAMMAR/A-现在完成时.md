# 现在完成时

- 1. 表示过去发生或已经完成的动作，对现在造成的影响或结果
- 2. 也可以表示过去某一时间开始并持续到现在的动作或状态。
- **多用于人生经验**
- `grammar 语法`
- `tenses 时态`

##### (一) 现在完成时的构成

- 主语 + 助动词 have/has + 过去分词

##### (二) 标志

- for + 时间段
- since + 时间点

```
标志词
---

1.
for + 表示一段时间的词
- for an hour 一个小时
- for a day 一天
- for a long time 很长一段时间
// 1. This large tree has been here for a long time. 这额大树在这里已经很久了
// 2. Mr Brown has taught us maths for three years. xxx已经教了我们三年的数学
// 3. the old man hasn't received his son's letter for a long time. 那位老人已经很久没有收到他儿子的信了

2.
since + 表示时间点的词/一般过去时从句
- since 9 o'click 从9点起
- since he left 自从他离开后
// I haven't seen Jhn since he left. 自从约翰离开后，我就没有见到过他了

3.
其他标志
- already 已经
- yet 还 尚 已经
- just 刚刚
- recently 进来
- lately 最近
- so far 到目前为止
- by now 到现在为止
- in the past year 在过去一年的时间里
// Grace has already come. xxx已经来了
// The cat has eaten up its food already. 那只猫已经把食物吃完了
// The cat hasn't eaten up its food yet. 那只猫没有把食物吃完
// Has the cat eaten up its food yet? 那只猫已经把食物吃完了吗？
// The bird has built its nest since last week. 那只鸟从上周起就在建鸟窝了

4.
长和 never yet 等连用，表示某个动作，某件事情从未发生过
// I have't been to the zoo yet. 我还没有去过动物园呢
// I have never seen a real panda. 我还从来没有见过真正的大熊猫

I have read this story. 我己经读了这个故事了
Adele and her brother have watered all the flowers. 阿黛尔和弟弟已经吧所有的花都浇完了
Adele has drawn two pictures. 阿黛尔已经画好两幅画了

They have left. 他们已经离开了
```

```
1.
have/has been 曾经去过某地
have/has gone 已经去某地了
// She has been to Beijing. 她曾经去过北京
// She has gone to Beijing. 她到北京去了 （ 她现在啃根已经在去北京的路上，或者已经到了北京，总之她现在不在这里 ）

2.
We have chatted for one hour. 我们已经聊了一个小时了
How long have they skipped? 他们跳绳多久了

3.
Max has sold his home. ---- 现在完成时，强调现在
Max sold his home. -------- 一般过去时，强调过去，强调过去的时间点

4.
I have been to Australia twice. 我去过澳洲两次
I went to australia early this year. 我今年年初的时候去了澳洲一趟
```
