##### Object clause. 宾语从句

- `Object 对象 物体 宾语`
- `clause 从句 子句`

```
1
宾语从句：
概念：充当 ( 宾语 ) 的句子，叫宾语从句
词性：宾语是 名词，即整个宾语从句充当 名词
位置：一般跟在 ( 动词 ) 或 ( 介词 ) 的后面

2
从句：
定义：凡是前面有 ( 连接词/引导词 ) 的句子，就是 ( 从句 )，也就是说，【 你看到一个句子前面有 ( 介词 )，这个句子就是 ( 从句 ) 】
句子: (句子构成) 至少是 ( 主谓 )


3
宾语从句考点：
-- 引导词
-- 语序
-- 时态
引导词:
-- 1. 什么都没有用: that
-- 2. 是否含义的词: if whether
-- 3. 特殊疑问句: what where when who / how much / how many
语序:
-- 陈述语序
时态:
-- 主过从过
-- 主现从任意
-- 主句真理从现


4
例子
// I know who he is.
// -
// Can you explain what the 【 passive voice 】 and the 【 active voice 】 are? 你能解释一下什么是被动语态和主动语态吗？
// -
// 1. Do you know where the nearest station is?
// 1. Do you know where she is right now?
// 1. Do you know when the next bus comes? 你知道下一班公共汽车什么时候来吗
// -
// 2. Can you tell me where the nearest bus stop is?
```
