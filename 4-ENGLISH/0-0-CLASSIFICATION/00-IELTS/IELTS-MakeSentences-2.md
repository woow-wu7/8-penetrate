-
- _【 be good at doing sth. 擅长做某事 】_
- _【 interrupt sb doing sth. 打断某人做某事】_
- _【 it's sb first time doing sth. 某人第一次做某事 】_
-
- `【 have a hard time doing sth. 做某事有困难 】`
-
- _【 prohibit doing sth. 禁止做某事 】_
-
- `【 finish doing sth. 完成做某事 】`
-
- _【 prevent sb from doing sth. 阻止某人做某事 】_
-
- `【 persist in doing sth. 坚持做某事 】`
-
- _【 mind doing sth. 介意做某事 】_
-
- `【 enjoy doing sth. 喜欢做某事 】`
-
- _【 be afraid of doing. 害怕做某事 ( 客观上造成的 ) 】_
- _【 be afraid to do sth. 害怕去做某事 ( 主观上的 ) 】_
